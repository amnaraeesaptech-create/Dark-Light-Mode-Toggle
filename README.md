# 🌙 Dark / Light Mode Toggle

A modern and responsive **Dark / Light Mode Toggle** built using **HTML, CSS, and JavaScript**. The project allows users to switch between light and dark themes with a smooth animation while saving their preference using **localStorage**.

## ✨ Features

* 🌞 Light Mode & 🌙 Dark Mode
* 🎨 Smooth color transition effects
* 💾 Saves theme preference with localStorage
* 🔄 Automatically loads the previously selected theme
* 🎯 Creative toggle switch with icons
* 📱 Fully responsive design
* ⚡ Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* localStorage API

## 🚀 How to Run

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Click the toggle switch to change themes.
5. Refresh the page to see your selected theme remain saved.
